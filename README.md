## 🌐 Deployment

The site deploys automatically to **GitHub Pages** on every push to `main` via the
workflow in [`.github/workflows/deploy.yml`](.github/workflows/deploy.yml).

**One-time setup:** in your repository, go to **Settings → Pages → Build and deployment**
and set the **Source** to **GitHub Actions**.
