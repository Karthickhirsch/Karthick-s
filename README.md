# Karthick S — Portfolio

A modern, responsive portfolio website for **Karthick S**, Senior Automation Test Engineer.

Built as a fast, dependency-free static site (HTML + CSS + vanilla JS) with a QA/testing-inspired theme — animated terminal, typing effect, count-up stats, scroll reveals, and a fully responsive layout.

## ✨ Features

- Terminal boot animation running a mock BDD regression suite
- Typing role animation & animated stat counters
- Skills, experience timeline, featured work, and contact sections
- Dark, gradient-accented UI with grid + glow background
- Mobile menu, scroll progress bar, active nav highlighting
- Accessible: honors `prefers-reduced-motion`
- Zero build step — just static files

## 🧰 Tech

`HTML5` · `CSS3` · `Vanilla JavaScript` · Google Fonts (Space Grotesk, JetBrains Mono, Inter)

## 🚀 Run locally

Just open `index.html` in a browser, or serve it:

```bash
# Python
python -m http.server 8000
# then visit http://localhost:8000
```

## 🌐 Deployment (GitHub Pages)

This repo auto-deploys to GitHub Pages via GitHub Actions (`.github/workflows/deploy.yml`)
on every push to `main`.

**One-time setup:**
1. Push this project to a GitHub repository.
2. Go to **Settings → Pages → Build and deployment**.
3. Set **Source** to **GitHub Actions**.

Your site will be live at `https://<your-username>.github.io/<repo-name>/`.
