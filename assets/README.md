# Assets

## Profile picture

To show your photo in the hero section, add an image here named exactly:

```
profile.jpg
```

- Recommended: square image (e.g. 400×400 or larger), face centered.
- Supported: rename to `.jpg` (or update the `src` in `index.html` if using `.png`/`.webp`).
- Until a photo is added, the hero shows a styled **"KS"** monogram placeholder automatically.
